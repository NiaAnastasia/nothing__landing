# Nothing Store Landing Page

A responsive landing page for the Nothing online store, built from a Figma mockup. 
The project showcases a modern product page with adaptive layout for all screen sizes.

## Demo
🔗 [Live Preview](https://NiaAnastasia.github.io/nothing__landing/)

## Design Reference
🖼 [Figma Mockup](https://www.figma.com/design/DtkQmQ797hk0nI4KfMi2Uq/BOSE-New-Version?node-id=6802-376&node-type=frame&t=X4j8GUjG2jPxGb3w-0)

## Technologies Used
- HTML, CSS/SASS
- BEM methodology for HTML structure
- Parcel for project build
- Grid and Flexbox for layout
- SASS mixins with media queries for responsive design

## Getting Started

1. Clone the repo
```
git clone https://github.com/NiaAnastasia/nothing__landing.git
cd nothing__landing
```

2. Install dependencies
```
npm install
# or
npm i
```

3. Run the project locally
```
npm start
```

## Features
- 🔥 Fully responsive design for all screen sizes
- 🎨 Built pixel-perfect from Figma mockup
